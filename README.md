🌙 Minha Calculadora Mágica - Sailor Moon

Olá! Seja bem-vindo(a) ao repositório da minha Calculadora Mágica, inspirada no estilo fofo do anime Sailor Moon! ✨

Este é um projeto que desenvolvi para treinar minhas habilidades iniciais em desenvolvimento web, aprendendo a conectar a estrutura, a beleza visual e o funcionamento de uma página de forma totalmente do zero.

🚀 O que eu aprendi a fazer neste projeto?

Dividi o projeto em três partes principais para entender como cada uma trabalha em equipe:

1. 🏗️ Estrutura (HTML)

Aprendi a organizar os elementos na tela: o visor, os botões e as imagens decorativas.

Aprendi a separar os botões normais (números) dos botões especiais (operadores de contas) para deixar tudo organizado.

Descobri como usar um sistema de "segurança" nas imagens: se um link de imagem da internet quebrar, meu site coloca uma imagem de reserva automaticamente no lugar!

🎨 2. Aparência e Estilo (CSS)

Pratiquei a criação de um efeito de vidro brilhante nos botões e no visor.

Aprendi a trabalhar com sombras para dar a impressão de que a calculadora brilha como um cristal mágico.

Criei uma animação suave que faz as decorações ao redor da calculadora flutuarem na tela.

⚡ 3. Funcionamento e Mágica (JavaScript)

Aprendi a capturar o clique do usuário em cada botão para mostrar os números na tela.

Desenvolvi a lógica matemática para somar, subtrair, multiplicar e dividir.

Criei um efeito super fofo de faíscas mágicas: toda vez que clicamos em um botão, surgem emojis mágicos (⭐, 💖, ✨, 🌙) que voam e desaparecem logo depois.

📂 Estrutura dos Arquivos

O projeto está dividido de forma bem organizada para facilitar a leitura:

index.html - Guarda toda a estrutura e textos da calculadora.

style.css - Contém as cores pastel, brilhos e animações do design.

script.js - Controla as contas matemáticas e as faíscas ao clicar.
